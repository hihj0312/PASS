정의

데이터를 읽고 저장하는 주변기기이다.
요즘은 잘 장착하지 않는 추세이다.

구분

 1.장착되는 위치에 따라
-내장형
-외장형
*가격은 내장형보다 외장형이 더 비쌈. 

 2.기능에 따라
cd 드라이브
-cd-rom: 읽는 기능만 가능
-cd-rw: 읽고 쓰는 기능 둘 다 가능
dvd 드라이브
-dvd-rom: DVD와 CD 읽는 기능 가능
-rw-combo: CD 쓰고 읽는 기능과 DVD 읽는 기능 모두 가능
-dvd 멀티 : cd, dvd 읽고 쓰는 기능 모두 다 가능
-블루레이 레코더 : 블루레이 플레이어에서는 cd, dvd 모두 재생되지만 역은 안됨.
